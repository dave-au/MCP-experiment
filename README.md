# MCP-experiment

Small MCP demo:
- `mcp_server_math.py` with tools `add` and `multiply`
- `mcp_server_shell.py` (allow-listed exec)
- `mcp_stdio_tap.py` (stdio tap with --pretty logging)
- `chatbot.py` (Langchain / langgraph agent using MCP tools)
- `llm_and_tools_log.py` (logger for comms between agent and LLM)
